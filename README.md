## Sistema para Imobiliárias - TCC

Este sistema foi desenvolvido como Trabalho de Conclusão de Curso (TCC) para o curso técnico em Informática para Internet. O projeto em questão é uma solução voltada ao dia a dia do **corretor de imóveis**, objetivando otimizar a gestão de imóveis, clientes e propostas comerciais.

##  Objetivo

Aplicar os conhecimentos adquiridos durante o curso no desenvolcimento de um sistema prático, simples, funcional e intuitivo que auxilie corretores de imóveis na organização das informações que envolvem seu trabalho.

## Tecnologias Utilizadas

- HTML5 / CSS3 / JavaScript;
- MySQL;
- Postgresql.

## Funcionalidades

- Cadastro de imóveis;
- Cadastro e gerenciamento dos usuários;
- Registro de propostas e negociações;
- Sistema de login (com diferentes níveis de acesso);
- Chatbot integrado à AI;
- Painel administrativo com estatísticas e histórico.
